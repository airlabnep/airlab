# Agri-Intelligence Research Lab (AIRL)

**Intelligence in Agriculture | Nepal**

## About

The Agri-Intelligence Research Lab (AIRL) is a research organization dedicated to advancing precision agriculture and geospatial technologies through drone-based research, computer vision, and AI-driven solutions.

## Website Features

- **Multi-page responsive website** built with HTML, CSS, and JavaScript
- **Research publications** database with filtering capabilities
- **Article repository** featuring peer-reviewed research content
- **Team profiles** showcasing research members
- **Project portfolio** displaying active research initiatives
- **Collaboration opportunities** and contact information
- **Course offerings** in precision agriculture and related technologies

## Getting Started

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/airl-website.git
   cd airl-website
   ```

2. Open `index.html` in your web browser to view the website locally

3. To make changes:
   - Edit HTML files for content updates
   - Modify `css/style.css` for styling changes
   - Update `js/script.js` for functionality changes

### Deploying to GitHub Pages

1. Push your changes to the `main` branch
2. Go to repository Settings → Pages
3. Select `main` branch as the source
4. Your site will be live at `https://airlabnep.github.io/airlab`
